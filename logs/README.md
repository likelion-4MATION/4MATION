# 데일리 스탠드업 로그

매일 아침 진행하는 데일리 스탠드업 미팅 내용을 기록하는 디렉토리.

---

## 📋 진행 방식

1. 미팅 전, 아래 [작성 항목](#-작성-항목)에 따라 오늘 날짜의 로그를 **미리** 작성해 GitHub에 업로드
2. 스탠드업 미팅에서 업로드된 내용을 짧게 공유
3. 미팅 종료 후, 논의 사항과 결과를 같은 파일에 추가로 정리

> 공유가 목적인 미팅이지 토론 중심 회의가 아니므로, 짧은 로그만 남기는 것을 원칙으로 합니다.

## ✏️ 작성 항목

**미팅 전**

- 전날 진행한 사항
- 금일 진행할 사항
- 현재 발생했거나 발생할 것으로 예상되는 이슈

**미팅 후**

- 논의 사항
- 논의 결과

## 📁 파일 규칙

- 파일명: `YYYY-MM-DD.md` (예: `2026-07-10.md`)
- 작성자: 1인 (담당자 고정)

## 📝 템플릿

```markdown
# YYYY-MM-DD 스탠드업

## 미팅 전

- 전날 진행한 사항:
- 금일 진행할 사항:
- 이슈 공유:

## 미팅 후

- 논의 사항:
- 논의 결과:
```
